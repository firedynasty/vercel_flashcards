Number Key Book Selection - Documentation
==========================================

Overview
--------
Pressing number keys (0-9) on the keyboard navigates to a Bible book
by its canonical order number (1-66). Digits accumulate in a buffer
with a 1.5-second timeout window, allowing both single-digit and
multi-digit input.

Source Location
---------------
- State:   src/BibleApp.js line 1021-1022
- Handler: src/BibleApp.js line 1045-1088
- Listener: src/BibleApp.js line 2097-2105

How It Works
------------
1. Press a digit key (0-9).
2. The digit is appended to an input buffer.
3. Each new digit resets the 1.5-second timer.
4. After 1.5 seconds of no further input, the accumulated number
   is parsed and matched to the canonical book list.
5. If the number is valid (1-66), the corresponding book is selected.
6. The buffer is cleared after processing.

Notes:
- Cmd/Ctrl + number is excluded (passed through for browser tab switching).
- Invalid numbers (0, 67+) are ignored with a console warning.
- Input fields and select dropdowns are excluded from key handling.
